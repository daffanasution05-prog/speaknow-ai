# SpeakNow AI

A premium AI-powered English speaking and learning assistant.

## Features
- **Free Chat**: Relaxed, open voice-enabled conversation.
- **Guided Mode**: Interactive training with AI-generated vocabulary hints.
- **Challenge Mode**: Fast-paced strict timer sessions.
- **Smart Dictionary**: Vocabulary builder with interactive tracking.
- **Interview Simulator**: Mock job interviews with feedback.
